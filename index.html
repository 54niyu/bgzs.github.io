使用Hugo搭建github静态博客

Hello World 哈哈哈哈哈哈哈哈好😁😆 最近突然想写博客了，想起github免费提供的一个静态空间，就想着去利用一下。刷知乎上看到了用golang写的hugo很适合 来生成静态页面. 下面就是自己折腾的这个新鲜的工具了。 第一步要安装hugo，得益于golang的编译方式，hugo的安装很简单，只需要去官网下载对应平台的二进制文件即可 Hugo 对于linux/unix系统，可以将它放置到/usr/loca/bin 以方便直接在terminal中敲命令 如果是macOs系统并且安装了Homebrew，只需要运行： brew update && brew install hua 当然，如果你熟悉golang的话，也可以选择下载源码安装 go get -v github.com/spf13/hugo 我是快乐的分割线
